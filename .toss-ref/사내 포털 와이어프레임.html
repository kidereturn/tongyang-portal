  • 폰트: Nanum Pen Script (제목), Gaegu (본문), Kalam (설명)
  • 컬러: 크림 페이퍼 + 먹색 · 노랑/파랑/핑크/초록 하이라이트

  • 플레이스홀더: 대각 줄무늬 + 설명 텍스트

✨ 각 옵션 핵심 차이

A · 사이드바 대시보드 — 가장 표준적. 업무 중심, 데이터 많을 때 유리.

B · 카드 허브 — 친근함. 기능 카드로 "뭐가 있는지"가 먼저 보임.

C · 매거진 — 콘텐츠(공지·웹툰·소식)가 풍부할 때 효과적.

D · 포스트잇 보드 — 캐주얼·재미 강조. 빙고/웹툰이 주연.
