## 슬라이드 5 — 담당자 사용법 (가장 중요)
1. **증빙관리** 메뉴 클릭
2. 본인 담당 통제활동 목록 표시
3. **업로드 버튼** 클릭 → 모달 창 열림
4. **드래그 앤 드롭** 또는 클릭으로 파일 선택 (PDF/Excel)
5. **자동 중간저장** — 파일 1개 추가할 때마다 자동 저장 ✨
6. 모든 모집단 업로드 완료 → **결재상신 버튼** 클릭
7. 승인자에게 자동 알림

> ⚠️ 결재상신은 **모든 모집단 항목이 업로드된 뒤에만** 가능합니다.

대본:
> "담당자분들이 가장 많이 쓰실 화면입니다. 증빙관리 가시면 본인 담당 활동이 나옵니다. 업로드 버튼 누르시면 팝업창이 열리고요. 파일을 드래그앤드롭으로 끌어다 놓으시면 됩니다. 신기한 건 — 파일 1개 올릴 때마다 자동으로 중간저장돼요. '저장 안 했다 잃어버릴까봐' 걱정하실 필요 없습니다."

> "모든 항목 업로드 끝났으면 우측 하단 '결재상신' 누르세요. 승인자한테 자동으로 가요."

## 슬라이드 6 — 담당자 자주 묻는 질문
**Q1. 잘못 올렸어요. 바꿀 수 있나요?**
- **교체** 버튼 → 새 파일 선택 → 중간저장 누르면 적용

**Q2. 삭제하고 다시 올리고 싶어요.**
- **삭제** 버튼 → 즉시 DB·Storage 에서 제거 → 새로 업로드

**Q3. 상신 후 수정할 수 있나요?**
- 승인 전에는 승인자가 반려해야 수정 가능
- 승인 후에는 관리자에게 '수정제출' 요청

**Q4. 다른 사람 증빙 볼 수 있나요?**
- ❌ 절대 불가능 (본인 담당 활동만 표시됨)

## 슬라이드 7 — 승인자 사용법
1. **증빙관리** 메뉴 → 본인 담당 통제활동 표시
2. 담당자가 상신한 활동에 **'승인'** / **'반려'** 버튼 노출
3. **승인** 클릭 → 즉시 처리
4. **반려** 클릭 → 사유 입력 → 담당자에게 자동 전달
5. 모달 안에서 모든 증빙 파일 **다운로드** 또는 **전체 ZIP 다운로드** 가능

대본:
> "승인자분들은 담당자가 상신한 건만 보입니다. 승인 / 반려 버튼이 나오면 누르시면 끝이에요. 반려하실 때는 사유 적어주셔야 담당자가 무엇을 수정할지 알 수 있어요."

---

## 슬라이드 8 — 관리자 사용법
1. **증빙관리** → 검토결과 드롭다운 (미검토 / 검토중 / 완료 / 수정제출)
2. **메모** 입력 가능 → 담당자·승인자에게 표시됨
3. **수정제출** 선택 → 모든 승인 자동 취소, 담당자 재상신 필요
4. **관리자 페이지** (`/admin`) → 전체 사용자, 통제활동, 진행률 등 마스터 관리
